TIPS FROM DEVELOPERS

HTML & CSS Tips

1.Always divide your html page into sections
<section id="section1">
  <div>
    <div>
      ......
    </div>
  </div>
</section>
2.Use your graphics in .svg extension rather than .jpg
<section id="section1">
  <div>
    <div>
      <img src="img.svg">
    </div>
  </div>
</section>
3.Use CSS for effects rather than images 
a:hover
{
  color:red;
}
4.Always give margins and paddings in percentages(%) 
It will make our webpage to look more floating and the
margins and paddings will get adjust themselves  
according to the sie of the device.

JavaScript Tips

1.Always place your scripts at the bottom of your page
<head>
</head>
<body>
  <script src="javascript.js"> 
  </script>
</body>
2.Always declare variable outside For statements
<script>
var container = document.getElementById('container');
for(var i = 0, len = someArray.length; i < len;  i++){
container.innerHtml += 'my number: ' + i;
}
</script>
3.Always make a separate page for javascript
Always make a separate .js file for javascript.
It makes your development better and let your
organize your project. It let you oraganize the project in
a heriarchy.
4.Use [] Instead of New Array()
<script>
var a = new Array();
a[0] = "Joe";
a[1] = 'Plumber';
</script>
