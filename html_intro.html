HTML

HTML stands for Hyper-Text Markup Language which is used to create or Markup the Web documents.

A markup language is set of markup tags.

HTML documents are described by HTML tags.

Each HTML tag describes different document content


It is an programming language of sorts.The instructions are compiled by the WWW browser.

HTML is basically a set of instructions and each such instruction is called an element.

The qualities of an element are called Attributes.These attributes control the behaviour of an element.

Why HTML?

HTML is at the heart of the Web.A language was required so as to create Webpages, so HTML been easy to write,easy to understand and highly portable was choosen.A HTML file contains instructions that any Web browser interprets and when these browsers display the file,it is known as a page on Web.

HTML Tags

HTML tags are keywords surrounded by angle brackets.

  • HTML tags normally come in pairs like <p> and </p>.
  • The first tag in a pair is the start tag, the second tag is the end tag
  • The end tag is written like the start tag, but with a forward slash inserted before the tag name

Usage of Tags:

<tagname> Contents Goes here....</tagname>
