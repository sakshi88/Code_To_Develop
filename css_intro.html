CSS

CSS stands for Cascading style-sheets which is used for the styling, of the Web documents.

CSS describes how HTML elements are to be displayed on screen, paper, or in other media

CSS saves a lot of work. It can control the layout of multiple web pages all at once

Each HTML tag describes different document content


Why CSS?

CSS provides you the way for styling of the web pages including designing, layout and optimization for various device sizes.

1. CSS solved the biggest problem!!

Previously when css was not introduced, developers has to write the styling code in HTML .

It became nightmare for the developers.

Developers of the large website has to write the color and font tags explicitly on the html page, which was very long and expensive process.

CSS removed the style formatting from the HTML page!



2.CSS saved a lot of overhead!!

You can put your css code in a separate single .css file and can link it to your HTML page .

It saves your overhead of writing the css code again and again.

With an external stylesheet file, you can change the look of an entire website by changing just one file!
