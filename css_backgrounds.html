CSS Selectors

CSS selectors are used to "find" the HTML element in a HTML page which you want to style.

1.The element selectors

The element selectors are used to select the elements based on the name

In the example below <p> is selected which will show effect(text color red) on every <p> tag on the html page

Example

p
{
color:20px;
}

2.The id selectors

The element selectors are used to select the elements based on the id of the element.

The id is given, to the element, using the id attribute of HTML

Id of the element must be unique on a HTML page.

To select an element with a specific id, write a hash (#) character, followed by the id of the element.

The following example shows the use of id selectors with CSS

It will apply CSS effects on the element with id="My_id"

Example

#My_id
{
color:20px;
height:90px
}

3.The class selectors

The class selectors are used to select the elements based on the class given to them.

The class is given, to the element, using the "class" attribute of HTML

class given to a element must be unique on a HTML page.

To select an element with a specific class, write a dot (.), followed by the class of the element.

In the example below, all HTML elements with class="center" will be red and center-aligned:

Example

.center
{
text-align: center;
color: red;
}

You can also select a specific element that can be affected by the class.

In the example below, only <p> elements with class="center" will be center-aligned:

Example

p.center
{
text-align: center;
color: red;
}

Grouping selectors

If you have diiferent selectors with same style definition like as follows:

Example

p
{
color:20px;
height:90px
}

h1
{
color:20px;
height:90px
}

h1
{
color:20px;
height:90px
}

It will be better to group them as to minimize the code

To group the selectors separate each of them with a comma.

Following example shows the grouping of selectors

Example

p,h1,h2


{

color:20px;


height:90px
}

CSS Comments

Comments are used to explain the code, and may help when you edit the source code at a later date.

A CSS comment starts with /* and ends with */. Comments can also span multiple lines:

Example

p {
color: red;
/* This is a single-line comment */
text-align: center;
}

/*
Multi line
comment
*/
